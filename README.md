
# Simple eShop

Online board game shop where you can choose, evaluate and buy goods.

## Run Locally

Clone the project

```bash
  git clone https://github.com/ValeryiaSharova/Simple-eShop
```

Go to the project directory

```bash
  cd Simple-eShop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
